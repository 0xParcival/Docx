Heading Tags

  • It adds a levels heading to the HTML document.

  • The <h1> to <h6> are HTML elements represent six levels of size heading.

  • <h1> is the highest level and <h6> is lowest level.

  • By default all heading elements are block level in layout.

NOTE :

Block level elements are starting on a new line and taking up the full width available in their containing block
(we will learn more in upcoming lessions).

Syntax

<!DOCTYPE html>
<html>
 <body>
    <h1>This is heading h1 </h1>
    <h2>This is heading h2 </h2>
    <h3>This is heading h3 </h3>
    <h4>This is heading h4 </h4>
    <h5>This is heading h5 </h5>
    <h6>This is heading h6 </h6>
    </body>
</html>
