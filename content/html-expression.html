Expression Tag

  • They are used to render the expression like H2O,a2+b2,etc..
  • There are two tags:
    • Superscript is used upperbound values. <sup> tag is used Ex: a<sup>2 is render as a2
    • Subscript is used lowerbound values. <sub> tag is used Ex: H<sub>2</sub>O is render as H2O

Syntax

<!DOCTYPE html>
<html>
<body>
<p>This is Superscript a<sup>2</sup>+b<sup>2</sup></p>
<p>This is Subscript H<sub>2</sub>O</p>
</body>
</html>
